Introduction to Application Development in Unity with Visual Programming

Tomaz Saraiva

In this workshop, we will learn how to develop applications in Unity using visual programming, an intuitive approach that allows the creation of interactive experiences without the need for prior programming knowledge. Through a step-by-step process, we will learn to use Unity's visual tools while creating a simple game. Throughout the workshop, you will discover how to apply this technique to various projects, opening doors to creating immersive and innovative experiences.

Workshop Program

  • Introduction to Unity and Visual Programming
  • Visual development tools in Unity (Visual Scripting)
  • Step-by-step creation of a simple game using visual programming
  • Implementation of interactions and behaviors in the game

Workshop Duration

3 hours

Maximum Participants

8 people

Ages

18 years and older

Participants should have basic computer skills and an interest in Digital Technologies and Game Development.

Requirements

Participants must bring their own computer with Unity installed (version 2022.3.49f1) and Visual Studio Code/Community.
The computer must meet the following minimum requirements:

  • Operating System: Windows 7 SP1+, 8, 10, only 64-bit versions; Mac OS X 10.13+; Ubuntu 16.04, 18.04, and CentOS 7
  • Graphics card: DX10-capable video card (shader model 4.0)
  • 8GB of RAM

The participants must bring a computer mouse.
