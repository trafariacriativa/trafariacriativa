WORKSHOPS

Introdução ao Desenvolvimento de Aplicações em Unity com Programação Visual

Tomaz Saraiva

Neste workshop, vamos aprender a desenvolver aplicações em Unity utilizando programação visual, uma abordagem intuitiva que permite criar experiências interativas, sem necessidade de conhecimento prévio em programação. Através de um processo passo a passo, iremos aprender a utilizar as ferramentas visuais do Unity enquanto criamos um jogo simples. Ao longo do workshop, descobrirás como aplicar esta técnica em diversos projetos, abrindo portas para criar experiências imersivas e inovadoras.

Programa do workshop

  • Introdução ao Unity e à Programação Visual
  • Ferramentas de desenvolvimento visual no Unity (Visual Scripting)
  • Criação passo a passo de um jogo simples utilizando programação visual
  • Implementação de interações e comportamentos no jogo

Duração do workshop

3 horas

Máximo de participantes

8 pessoas

Idades

A partir dos 16 anos

Os participantes deverão ter conhecimentos básicos de informática e interesse em Tecnologias Digitais e Desenvolvimento de Jogos.

Requisitos

Os participantes devem trazer o seu próprio computador com o Unity instalado (versão 2022.3.49f1) e Visual Studio Code/Community.
O computador deve cumprir os seguintes requisitos mínimos:

  • Sistema Operativo: Windows 7 SP1+, 8, 10, somente versões de 64 bits; Mac OS X 10.13+; Ubuntu 16.04, 18.04 e CentOS 7.
  • Placa gráfica: Placa de vídeo com recursos DX10 (shader model 4.0)
  • 8GB de memória RAM

Os participantes devem trazer um rato de computador.
